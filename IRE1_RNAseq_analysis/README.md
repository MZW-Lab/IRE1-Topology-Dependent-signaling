# IRE1alpha Topology-Dependent Transcriptomics: Analysis Code

Code repository accompanying:

> **A topology-dependent IRE1alpha signaling axis generates a cryptic XBP1 isoform present in human cancers**
> Jose Roberto Rodrigues Reyes, Collin Vergel De Dios, Debalina Datta, Taivan Batjargal, Surenna Pecchia, Feroz R. Papa, Maxwell Z. Wilson
> *Nature Chemical Biology* (2026)

This repository contains the code to reproduce all RNA-seq figures from the time-series optogenetic experiment, including DESeq2 differential expression with spline fitting, UMAP sample projection, genome-wide heatmap with k-means clustering, RIDD target time-series, cell-cycle gene set analysis, and GO over-representation analysis.

## Experimental design

HEK293T cells stably expressing either ER-opto-IRE1alpha (ER-tethered) or Cyto-opto-IRE1alpha (cytoplasmic, transmembrane domain deleted) were illuminated continuously using the OptoPlate system. Samples were collected every 15 minutes over a 345-minute time course, giving 24 timepoints per condition and 48 samples total. Each timepoint is a single sample with no biological replicates per timepoint; dispersion is estimated across the time course by DESeq2 with spline fitting.

## Running the analysis

All scripts are designed to be run from the **repository root** directory. Run them in order, as Script 01 generates intermediate files (`DESeq2_*_results.csv`, `vst_combined.csv`) that are required by all downstream scripts.

```bash
Rscript R/01_DESeq2_spline.R    # ~5 min; generates DEG tables and VST matrix
Rscript R/02_UMAP.R             # ~1 min; requires vst_combined.csv
Rscript R/03_heatmap_3B.R       # ~2 min; requires vst_combined.csv and cluster_assignments.csv
Rscript R/04_RIDD_targets_3C.R  # <1 min; requires vst_combined.csv
Rscript R/05_cell_cycle_3E.R    # <1 min; requires vst_combined.csv and PanelA_gene_sets.csv
Rscript R/06_ORA_S4CD.R         # ~3 min; requires DESeq2_*_DEGs.csv
```

Output PDFs are written to `figures/`.

## Methods summary

### DESeq2 with natural cubic spline (Script 01)

Raw counts were loaded and rows with missing gene symbols were removed (617 unnamed genes per matrix). Genes with fewer than 10 counts across all samples were filtered out prior to modeling. For each condition separately, a DESeqDataSet was constructed with a natural cubic spline design (`~ ns(time, df=4)`) and tested against a reduced intercept-only model using the likelihood ratio test (LRT). This approach identifies genes whose expression changes significantly over the time course without assuming a particular functional form. Significance threshold: adjusted p < 1x10^-3 (Benjamini-Hochberg correction).

A combined VST-normalized matrix across all 48 samples was computed using `vst(blind=TRUE)` for use in UMAP and heatmap visualization.

### UMAP (Script 02)

VST-normalized expression was computed on the combined 48-sample matrix. The top 500 most variable genes (by row variance) were selected and UMAP was run using the `uwot` package with `n_neighbors=15`, `min_dist=0.3`, Euclidean distance, and a fixed random seed (42) for reproducibility.

### Genome-wide heatmap (Script 03)

Log2 fold changes relative to t=0 were computed as the difference in VST-normalized values: log2FC = VST(t) - VST(t=0). The pre-computed k-means cluster assignments in `cluster_assignments.csv` are used directly without re-clustering, ensuring exact reproducibility of the published figure. The color scale is capped at +/-2 log2FC. Cluster biological identities were verified by GO enrichment and are documented in the script comments.

### IRE1alpha expression normalization (Script 04)

To correct for the approximately 4-fold difference in total IRE1alpha protein abundance between constructs, log2FC values were divided by per-condition normalization factors derived from western blot quantification (anti-total-IRE1alpha normalized to beta-actin). Normalization factors: ER = 0.321, Cyto = 1.679. See manuscript Methods for the full derivation.

### Cell-cycle gene sets (Script 05)

Mean log2FC +/- SEM was computed across all genes in each cell-cycle phase gene set (G1/S, S, G2/M, M/G1, Arrest) at each timepoint. Only genes with `status == "used"` in `PanelA_gene_sets.csv` are included; two genes were excluded due to low expression at t=0.

### GO over-representation analysis (Script 06)

Gene symbols were converted to Entrez IDs using `clusterProfiler::bitr()`. GO Biological Process ORA was performed with `enrichGO()` against a background universe of all genes tested in DESeq2. Redundant GO terms were reduced using `simplify()` with a similarity cutoff of 0.7. Significance threshold: adjusted p < 0.05 (BH correction).

## Notes on reproducibility

- DESeq2 results may differ slightly from the manuscript values depending on R and Bioconductor versions. The DEG counts produced here (Cyto: ~5,346; ER: ~645) match the manuscript's reported values (5,319 and 651) within rounding.
- UMAP coordinates are stochastic. Set the random seed before running to reproduce the exact layout shown.
- The k-means cluster assignments in `cluster_assignments.csv` are the exact assignments used in the manuscript. Running k-means fresh will produce biologically equivalent but not numerically identical cluster memberships due to random initialization.
- All figures are output as editable PDFs with TrueType fonts (useDingbats=FALSE).
